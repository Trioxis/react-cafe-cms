# Development Readme

## Install locally
Clone and install locally:
```shell
git clone https://github.com/Trioxis/react-cafe-cms.git
cd react-cafe-cms
yarn install
```

## Run tests
Run [Jest](https://facebook.github.io/jest/) tests in watch mode.
```shell
# Use '--' if you're using yarn before v1.0
yarn test --watch
```
